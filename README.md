# 📊 Student Performance Analysis Dashboard

An interactive **Power BI Dashboard** developed to analyze and visualize student academic performance. This project transforms raw student data into meaningful insights through interactive visualizations, KPI cards, and dynamic filters. The dashboard enables users to evaluate student performance in **Mathematics, Reading, and Writing** while analyzing the influence of **gender, ethnicity, parental education level, and test preparation status**.

---

## 📖 Project Overview

The **Student Performance Analysis Dashboard** is a Business Intelligence project built using **Microsoft Power BI**. The primary objective of this project is to transform raw educational data into meaningful visual insights that help users understand student performance from multiple perspectives.

The dashboard provides an intuitive and interactive interface that allows users to compare subject-wise scores, analyze demographic trends, identify top-performing students, and evaluate the impact of parental education and test preparation on academic performance. By combining data visualization techniques with interactive filtering, the dashboard supports effective analysis and data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze overall student academic performance.
- Compare Mathematics, Reading, and Writing scores.
- Study performance across different genders and ethnic groups.
- Evaluate the influence of parental education level on student performance.
- Analyze the impact of test preparation on academic outcomes.
- Identify the top-performing students based on overall scores.
- Develop an interactive dashboard for efficient data visualization and reporting.

## 📂 Dataset Information

The project uses a dataset containing academic records of **100 students**.

### Dataset Fields

- Student ID
- Gender
- Ethnicity
- Parental Education Level
- Test Preparation Status
- Mathematics Score
- Reading Score
- Writing Score

---

## 🧹 Data Preparation

The following preprocessing steps were performed before developing the dashboard:

- Imported the Excel dataset into Microsoft Power BI.
- Verified and corrected data types.
- Checked for missing values and duplicate records.
- Cleaned and transformed the data using Power Query.
- Created calculated measures and KPIs using DAX.
- Designed interactive visuals and slicers for dynamic analysis.

## 🔍 Key Insights

- Students who **completed the test preparation course** achieved higher average scores compared to those who did not.
- **Ethnicity Group A** recorded the highest average academic performance among all groups.
- Female students performed slightly better overall than male students.
- Mathematics recorded the highest average score among the three subjects.
- The dashboard identifies the **Top 5 Performing Students**, making it easy to recognize high achievers.
- Interactive filters allow users to explore student performance across different categories and gain deeper insights.
